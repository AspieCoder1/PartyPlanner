# Register user
Registers a new user
- URL: `/register`
- Method: `POST`
- Auth required: NO

## Parameters
This endpoint does not have any query parameters

## Request body
|Key         |Description                                         |
|------------|----------------------------------------------------|
|email       | Email of the new user                              |
|username    | Username of new user                               |
|password    | Password of the new user                           |

## Headers
This endpoint does not require any custom headers to be set.

## Success Response
Code: `200 OK`
### Example Response
```json
    {
      "id": "6036b50bd562097c0bdf6ada",
      "userName": "test"
    }
```

## Errors
- `400`, in the case the of an error in email or password is not found.
- `404`, email not found.
- `401`, passwords is incorrect.
- OR in the case of a server error you would get a `500` error in the form of `{code: 5xx, body: 'Oh snap, something went wrong on our end.'}`

## Example request
```js
var axios = require("axios");
const user = {
      email: 'test@test.com',
      username: 'test',
      password: 'abcdefghht'
};

axios.post('/api/users/registers/', user).then(res => {
    console.log(res.body);
}).catch(err => {
    console.log(err)
})
```

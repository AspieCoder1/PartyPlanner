# Register user
Registers a new user
- URL: `/login`
- Method: `POST`
- Auth required: NO

## Parameters
This endpoint does not have any query parameters

## Request body
|Key         |Description                                         |
|------------|----------------------------------------------------|
|email       | Email of the new user                              |
|password    | Password of the new user                           |

## Headers
This endpoint does not require any custom headers to be set.

## Success Response
Code: `200 OK`

## Errors
- `400`, in the case the of an error in either validation or the email already exists.
- OR in the case of a server error you would get a `500` error in the form of `{code: 5xx, body: 'Oh snap, something went wrong on our end.'}`

## Example request
```js

const user = {
      email: 'test@test.com',
      password: 'abcdefghht',
};

axios.post('/api/users/registers/', user).then(res => {
    console.log(res.body);
}).catch(err => {
    console.log(err)
})
```

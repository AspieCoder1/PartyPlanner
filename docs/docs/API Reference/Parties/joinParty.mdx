# Join Party
Join a party via link/public party button
- URL: `/join/:id`
- Method: `POST`
- Auth required: NO

## Parameters
|Parameter|Description                                      |
|---------|-------------------------------------------------|
|id       | The party id of the party the user wants to join|

## Request body
|Key         |Description                                         |
|------------|----------------------------------------------------|
|name        | Name of the new party                              |
|username    | Username of the user who wants to joinh the party  |

## Success Response
Code: `200` with the party the user joined.
### Example Response
This is the case where the id is `507f1f77bcf86cd799439011`. The username parameter is 'jonhSmith1'
```json
{
  "party": {
    "_id": "507f1f77bcf86cd799439011",
	"attendeesID": "[..., 'johnSmith1']"
    ...
  }
}
```

## Errors
- `400`, in the case the party is not found. This would look like `{code: 400, error_msg: 'No party of this id exists'}`.
- OR in the case of a server error you would get a `500` error in the form of `{code: 5xx, error_msg: 'Oh snap, something went wrong on our end.'}`

## Example request
```js
var axios = require("axios");
axios.post('/api/parties/join/507f1f77bcf86cd799439011').then(res => {
    console.log(res.body)
}).catch(err => {
    console.log(err)
})
```
